BACKEND

¿QUE ES BACKEND?

Las tecnologías de backend son fundamentales en el desarrollo de proyectos de software del día a día.
El desarrollo del back-end, a menudo conocido como desarrollo "del lado del servidor", consiste en hacer que todo el funcionamiento interno de un sitio web o aplicación sea invisible para el usuario (SILK, 2022).
Es la parte del desarrollo web que se encarga de que toda la lógica de una página web funcione, al tratarse del conjunto de acciones que pasan en una web, no vemos como es la interacción con el servidor.
Un Back-end es una capa de entras o código de programa al que los usuarios no tienen acceso al ser un aspecto que el usuario no podrá ver en su navegador, entre ellos el servidor y base de datos, siendo información que se cambia o actualiza diariamente (Souza, 2020).
Encargado de desarrollar back-end se pasa el día entero interactuando con diferentes y varias códigos de programación, siendo meticulosa y cuidadoso en su trabajo solo una pequeña falla y la página web se puede tumbar sola (Arjonilla, 2016).

¿Quién es un desarrollador de back-end?

Es el encargado de la arquitectura del sitio web, las secuencias de comandos y el código que sirve de puente entre la base de datos del sitio web y el navegador del usuario siendo este un desarrollador de software especializado en diseñar, construir y probar la lógica de back-end/negocio de una aplicación (Souza, 2020).).

Funciones de los desarrolladores del Back End

Los desarrolladores web de Back-end se aseguran de que los datos y sistemas solicitados por el software funcionen de manera eficiente.
